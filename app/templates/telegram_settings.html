Интеграция Telegram

Опционально: укажите токен бота и chat id группы. Бот также умеет показывать ID чата и запускать проверки.

После сохранения бот начнёт опрашивать Telegram.
Если группа с топиками — укажите topic thread id или включите через /events_on прямо в нужном топике.

Команды бота

  • /start — справка и показ ID чата.
  • /menu — открыть главное меню бота с кнопками.
  • /chatid — показать ID текущего чата (для вставки в настройки).
  • /run — открыть инлайн-меню: выбрать агента, затем задачу.
  • /probe_offline — поставить проверку/heartbeat-пробу для offline агентов.
  • /events_on и /events_off — вкл/выкл дублирование событий из бота.
